### `img/`

This directory contains all the images used in the project, organized into subdirectories for banners, product grids, icons, logos, and product images.

### `src/`

This directory contains all the CSS files, organized into base styles, resets, variables, and components.

- **`base.css`:** Basic styles for the body, buttons, and other global elements.
- **`reset.css`:** A CSS reset to ensure consistent styling across different browsers.
- **`style.css`:** The main stylesheet for the project.
- **`variables.css`:** Contains CSS variables for colors, fonts, etc.

### `src/componentes/`

This directory contains the CSS files for the different components of the website, such as the header, footer, hero section, etc.

- **`footer.css`:** Styles for the footer component.
- **`header.css`:** Styles for the header and navigation.
- **`hero.css`:** Styles for the hero section.
- **`product-category.css`:** Styles for the product category cards.
- **`product-grid.css`:** Styles for the featured products grid.

### `index.html`

The main HTML file for the website.
